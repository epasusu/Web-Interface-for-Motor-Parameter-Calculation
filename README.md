В текущей версии имеется 4 расчета:
1. Расчет статических и энергетических характеристик АД с короткозамкнутым ротором;
2. Расчет статических и энергетических характеристик АД с фазным ротором;
3. Расчет сопротивлений обмоток АД с короткозамкнутым ротором;
4. Расчет сопротивлений обмоток АД с фазным ротором.
